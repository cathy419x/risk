# PSC Risk Calculator

一个可直接部署到 GitHub 的单文件网页计算器。  
A single-file web calculator ready for GitHub deployment.

## 文件
- `index.html`：主页面

## 用法
1. 新建一个 GitHub 仓库
2. 上传 `index.html`
3. 在仓库设置中启用 GitHub Pages
4. 选择从主分支部署

## 功能
- 中文 / English / 双语 切换
- 基于 Logistic 回归模型计算 PSC 风险
- 实时显示风险概率、风险等级、系数贡献和公式说明
